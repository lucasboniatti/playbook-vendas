# Playbook de Vendas WhatsApp

Aplicação React com dois playbooks completos para vendas de sites via WhatsApp.

## 📚 Conteúdo

- **Playbook Completo**: Guia detalhado com 7 etapas de vendas, scripts, objeções e regras de ouro
- **Script IA**: Fluxo automático com 6 etapas para automação com IA

## 🚀 Como usar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra no navegador: `http://localhost:5173`

## 🔨 Build para produção

```bash
npm run build
```

## 📦 Deploy na Vercel

1. Faça push do código para GitHub
2. Conecte seu repositório na [Vercel](https://vercel.com)
3. Clique em "Deploy"

A aplicação estará disponível em uma URL pública automaticamente!
